## Surface Pro Kiosk Setup

Recommended browser settings:

- Use Microsoft Edge.
- Open the GitHub Pages URL.
- Set portrait orientation in Windows display settings.
- Use Windows assigned access / kiosk mode once the app is stable.
- Disable sleep while plugged in.
- Keep screen timeout long enough for household use.

## Daily Use

- Use Today for the walk-by dashboard.
- Use the plus button for quick shared tasks.
- Use Approvals for allowance and paid job review.
- Use Manage to adjust chores and dinner.
- Use Start Brand New Week Loop when weekly chores need to reset.

## Recovery

If the screen looks wrong:

1. Refresh the browser.
2. Check whether the top status says `System Live`.
3. If it says local/offline, check internet.
4. If data is stale on one device, close and reopen the browser.
5. If Firebase data is corrupted, restore from GitHub history or Firebase backup/export if available.
