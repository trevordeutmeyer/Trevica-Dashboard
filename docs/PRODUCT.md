# Product Notes

## Goal

Create a family command center that is useful at a glance and easy to manage from the same screen.

## Main Kiosk View

The first screen should answer:

- What is happening today?
- What chores or tasks are still open?
- Who has completed their work?
- Are any approvals waiting?
- What is for dinner?
- What is on the calendar?

## Core Workflows

- Add a quick shared task.
- Mark chores complete.
- Request approval for paid jobs.
- Parent approves or denies items.
- Start a new weekly loop.
- Update dinner plan.
- Review recent history.

## Design Direction

- Portrait-first.
- Large touch targets.
- High contrast.
- Readable from several feet away.
- No dense admin controls on the first screen.
- Management tools can be one tap deeper.

## Open Decisions

- Final family member names and chore lists.
- Whether parents need separate logins.
- Whether kids should use the same kiosk only or also phones.
- Whether allowance should reset weekly automatically.
- Whether calendar should be public embed, private family calendar, or a custom event list synced from Firebase.
